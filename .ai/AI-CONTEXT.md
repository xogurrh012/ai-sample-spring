# .ai/

## 목적
AI 에이전트(Gemini CLI 등)가 프로젝트를 이해하고 일관된 코드를 작성할 수 있도록 돕는 지능화 레이어.

## 주요 파일
| 파일명 | 설명 |
| --- | --- |
| project-context.md | 프로젝트의 전체 맥락과 비즈니스 로직 정의 |
| rules/code-convention.md | 프로젝트 공통 코드 스타일 및 패키지 구조 규칙 |

## 하위 디렉토리
- `rules/` - 코딩 스타일 및 기술 제약 사항
- `skills/` - `/deep-interview`, `/deepinit` 등 확장 기능 정의

## AI 작업 지침
- 이 폴더 내의 파일들을 코드 수정 전 반드시 먼저 읽어야 한다.
- `AI-GUIDE.md`의 스킬 목록을 업데이트할 때는 이 폴더의 `skills/` 내용을 스캔한다.
